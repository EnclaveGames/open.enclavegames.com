Open Enclave Games

All the open source projects created by Andrzej Mazur from Enclave Games are listed here, on the open.enclavegames.com website. Feel free to check them out, learn from them and send pull requests if you have any feedback.

  • Enclave Phaser Template
  • Intro to 3D Games
  • Gamepad API Content Kit
  • Gamedev Phaser Content Kit
  • Gamedev Canvas workshop
  • Cyber Orb
  • Monster Wants Candy
  • Other projects

If you like what I'm doing and gonna use any of the available source code in your project, or just appreciate open gamedev, you can say "thank you" by buying me a beer. Cheers!

Enclave Phaser Template

  • GitHub source code: github.com/EnclaveGames/Enclave-Phaser-Template
  • Online demo: enclavegames.github.io/Enclave-Phaser-Template/
  • Related article: dev.end3r.com/2015/12/enclave-phaser-template-is-open-sourced/

Fully mobile responsive, portrait and landscape compatible template for HTML5 games written using the Phaser game engine, with helper functions, organized in states.

Intro to 3D Games

  • GitHub source code: github.com/end3r/MDN-Games-3D/
  • Online demos: end3r.github.io/MDN-Games-3D/
  • Related articles: Basics of building 3D games on the Web

MDN Games introductory tutorials with articles and demos on how to start building 3D games using Three.js, PlayCanvas, BabylonJS, but also A-Frame, shaders.

Gamepad API Content Kit

  • GitHub source code: github.com/end3r/Gamepad-API-Content-Kit
  • Online demo: end3r.github.io/Gamepad-API-Content-Kit/demo/demo.html
  • Related article: developer.mozilla.org/en-US/docs/Games/Techniques/Controls_Gamepad_API

Gamepad API Content Kit is where you can find all the resources about the Gamepad API, and a game demo utilizing this API.

Gamedev Phaser Content Kit

  • GitHub source code: github.com/end3r/Gamedev-Phaser-Content-Kit
  • Online demo: end3r.github.io/Gamedev-Phaser-Content-Kit/demos/lesson16.html
  • Related article: developer.mozilla.org/en-US/docs/Games/Workflows/2D_breakout_game_Phaser

This Content Kit contains resources on how to start building games with Phaser.

Gamedev Canvas workshop

  • GitHub source code: github.com/end3r/Gamedev-Canvas-workshop
  • Online demo: breakout.enclavegames.com/lesson10.html
  • Related article: developer.mozilla.org/en-US/docs/Games/Workflows/2D_Breakout_game_pure_JavaScript/

This workshop contains resources on how to start building games in pure JavaScript and render them on Canvas.

Cyber Orb

  • GitHub source code: github.com/EnclaveGames/Cyber-Orb
  • Online demo: orb.enclavegames.com
  • Related articles: developer.mozilla.org/en-US/docs/Games/Workflows/HTML5_Gamedev_Phaser_Device_Orientation

Simple HTML5 mobile demo game built with Phaser that uses the Device Orientation and Vibration APIs.

Monster Wants Candy

  • GitHub source code: github.com/EnclaveGames/Monster-Wants-Candy-demo
  • Online demo: candy-demo.enclavegames.com
  • Related articles: gamedevelopment.tutsplus.com/tutorials/getting-started-with-phaser-building-monster-wants-candy--cms-21723

Simple HTML5 game created with Phaser utilizing states and built step by step.
